# 📚 英语打卡小站

一个轻量级的英语学习打卡平台，帮助用户养成每日英语学习习惯。

## ✨ 功能特点

- 📖 **今日学习** - 每日单词、句子、打卡功能
- 🏆 **排行榜** - 学习打卡排名，正向激励
- 📊 **我的统计** - 打卡日历、热力图、趋势分析
- 👤 **个人中心** - 用户信息、密码修改
- 🔧 **管理后台** - 用户管理、数据统计
- 📱 **响应式设计** - 支持手机、平板、电脑

## 📱 页面说明

### 今日学习页 (/)

- 欢迎语 + 连续打卡徽章
- 每日单词卡片（含发音功能）
- 每日句子卡片
- 打卡按钮（带彩带动画）

### 排行榜页 (/rank)

- 前三名展示（金银铜特殊样式）
- 完整排行榜列表
- 当前用户高亮显示

### 我的统计 (/stats)

- 统计概览卡片（连续天数、总天数、本月天数）
- 打卡日历热力图
- 最近30天打卡趋势图

### 个人中心 (/profile)

- 用户信息展示
- 密码修改

### 管理后台 (/admin)

- 用户列表管理
- 添加/编辑/删除用户
- 重置密码
- 搜索功能

## 💾 数据存储

本应用使用浏览器 localStorage 存储数据，包括：

- 用户数据
- 打卡记录
- 学习内容
- 管理员信息

⚠️ **注意**：数据存储在浏览器本地，清除浏览器数据会导致数据丢失。

## 🔧 技术栈

- **前端**：纯 HTML + CSS + JavaScript
- **存储**：localStorage
- **UI框架**：无（原生CSS）
- **图表**：SVG绘制
